# 5月直播间短视频投放复盘

这是一个可直接用于 GitHub Pages 发布的静态网页目录。

## 发布方式

1. 新建一个 GitHub 仓库。
2. 上传本目录下的全部文件到仓库根目录：
   - `index.html`
   - `.nojekyll`
3. 进入仓库 `Settings` → `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/root`。
6. 保存后等待 GitHub 生成 Pages 链接。

如果你想放在已有仓库，也可以把这两个文件放进仓库的 `docs/` 目录，然后 GitHub Pages 目录选择 `/docs`。
