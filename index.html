素材重合结论

  • TOP引流、TOP成交、TOP跑量的主力素材基本重合，说明5月不是“拉人素材”和“卖货素材”分离，而是同一批素材同时完成引流、成交和放量。
  • 最强综合素材是“谁都不缺衣服，缺的是换装的好心情”：进房 28,943、GMV ¥289,790、消耗 ¥12,147、ROI 23.86，三项能力都强。
  • “怕热的姐妹，一定要入这款凉感面料”成交效率更强，ROI 27.20，适合重点复刻“夏季痛点 + 面料证明”结构。

类目分析

  • 针织衫：TOP10里占8条，是5月核心类目；既能引流，也能成交和跑量。
  • 针织短袖：CTR 7.17%，进房 13,595，适合作为夏季舒适/凉感补充类目。
  • 合集：点击强、引流强，但用户进房后购买路径更分散，需要拆到明确单款或明确类目承接。
  • 衬衫/防晒衬衫：未进入TOP综合素材主力，说明当前防晒衬衫需要补更强的首屏卖点和直播间承接。

高效素材模型

5月最有效的视频不是单纯“高点击”或“高成交”，而是“夏季痛点 + 明确款式 + 直播间/门店真实展示 + 面料证明 + 同款承接”。

  • 最能拉人：夏季痛点明确、标题有情绪或利益点、CTR能拉起的素材。
  • 最能卖货：面料、版型、显瘦、凉感证明充分，直播间能直接承接同款的素材。
  • 最容易跑量：消耗放大后ROI仍能稳在20以上，且进房和订单同步增长的素材。

天数分层

  • 1-3天：完成初筛，重点看CTR、进房、成交是否跑出基础反馈；不达标及时停投或重剪。
  • 3-5天：进入放量判断期，如果ROI高于保本线，且进房和成交同步提升，可加预算测试。
  • 5-7天：判断素材是否具备稳定承接能力，适合开始复刻同结构新素材。
  • 7天以上：说明素材已经具备持续投放能力，但要重点监控CTR和CVR是否衰退。

输出结论

5月素材判断不需要等到10天，正常应在1-3天完成初筛、3-5天决定是否加预算、5-7天判断是否复刻。当前能跑到10天以上的素材，说明已经通过多轮投放验证；其中“压力可以放一放，快乐不能缺席”投放27天、覆盖174个直播间，仍保持ROI 22.41，是长周期素材代表；“怕热的姐妹”和“换装好心情”虽然投放天数略短，但ROI和成交效率更强，适合继续复刻放量。

五、问题复盘

问题1：过度依赖针织衫单品

现象

针织衫消耗 ¥72,282，占短视频消耗大头；D18628693 单款消耗 ¥45,486，GMV ¥1,075,624。

原因分析

针织衫同时承接夏季凉感、显瘦、通勤、面料舒适多个卖点，短视频表达空间大，直播间也容易承接。

优化方案

继续放量针织衫，但要同步复刻衬衫、针织短袖、半身裙的高ROI结构，避免单款疲劳。

问题2：薄荷衫ROI低于保本

现象

薄荷衫消耗 ¥1,355，GMV ¥11,449，ROI 8.45，低于保本ROI 12。

原因分析

当前薄荷衫更像出游/情绪种草，成交证明不足，直播间承接可能弱。

优化方案

暂停放量，改为小预算测新脚本：显白、凉感、防晒/通勤、不同身材上身对比。

问题3：高CTR素材不一定高成交

现象

合集素材CTR 10.44%，进房15,598，但CVR 2.56%，低于针织衫主力素材。

原因分析

活动/合集能拉人，但用户进房后购买路径分散，缺少明确同款承接。

优化方案

高CTR低CVR素材用于引流可以保留，但直播间必须把主推款、价格机制、尺码承接讲清楚。

七、一页总结（老板版）

5月核心结论

1. 最强引流素材：谁都不缺衣服，缺的是换装的好心情，进房 28,943。 2. 最强成交素材：谁都不缺衣服，缺的是换装的好心情，GMV ¥289,790；“怕热姐妹”紧随其后，GMV ¥280,632且ROI 27.20。 3. 最强跑量素材：谁都不缺衣服，缺的是换装的好心情，消耗 ¥12,147，ROI 23.86。 4. 最优货品：针织衫D18628693，款式GMV ¥1,075,624，ROI 23.65。 5. 最大问题：货品和素材结构集中度高，薄荷衫等非主力货品暂未跑出成交效率。

6月核心动作

继续做：

  • 针织衫D18628693放量
  • 凉感、透气、显瘦、弹力证明型素材

停止做：

  • 低ROI薄荷衫原脚本
  • 没有同款承接的泛活动素材

重点复刻：

  • 运动作证明：拉伸、不透、风吹、贴肤
  • 人群痛点：怕热、通勤、显瘦背薄、小个子

重点测试：

  • 半身裙高ROI素材扩量
  • 衬衫通勤/防晒脚本
  • 薄荷衫新卖点重拍

预算倾斜：

  • 50%给已验证主力针织衫
  • 30%做新素材和新货品测试
  • 20%守稳定成交计划

复盘核心回答

1. 什么视频最能拉人？

夏季痛点明确、首屏有动作证明、能让用户进直播间确认尺码和上身效果的视频最能拉人。代表素材是“谁都不缺衣服，缺的是换装的好心情”。

2. 什么视频最能卖货？

能把痛点、面料证明、上身效果和直播间承接串起来的视频最能卖货。代表素材是“怕热的姐妹，一定要入这款凉感面料”和“简约大气不挑人，上班穿得体又高级”。

3. 什么视频最容易跑量？

大人群痛点 + 强视觉证明 + ROI守得住的视频最容易跑量。针织衫D18628693相关素材是5月最稳定跑量模型。

4. 什么货最值得继续投？

针织衫D18628693最值得继续投；半身裙ROI高，适合扩大测试；衬衫可保留但需要优化脚本效率；薄荷衫暂缓放量。

5. 6月预算应该投给谁？

预算优先给针织衫D18628693及其复刻素材，其次给针织短袖/衬衫的夏季通勤脚本，再用小预算测试半身裙和薄荷衫新脚本。
